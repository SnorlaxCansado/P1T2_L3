Repository: hsfinance

Overview

Hsfinance is a Stock Analysis Report Generator that combines various data sources to produce comprehensive reports for selected stock tickers. It leverages multiple APIs and AI tools to gather financial, geopolitical, and sector-specific news, creating insightful reports in both text and PDF formats.

Main Features

  • Selects complementary tickers for comparative analysis.
  • Collects stock data from Yahoo Finance and news using SERPER and Jina AI APIs.
  • Generates AI-powered insights on stock performance, geopolitical impacts, and sector-specific context.
  • Produces a well-formatted PDF report with all relevant data and analysis.

Setup Instructions

  1. Clone the repository using git clone.
  2. Create a virtual environment with python -m venv venv.
  3. Activate the environment and install dependencies with pip install -r requirements.txt.
  4. Set environment variables in a .env file (API keys for OpenAI, SERPER, and Jina Reader).
  5. Run the script using python src/main.py <ticker> --articles <number> --period <period>.

Technologies Used

  • Python (pandas, numpy, logging, ReportLab, fpdf2)
  • APIs: OpenAI GPT, SERPER, Jina AI Reader, Yahoo Finance
  • Data visualization with Matplotlib

Repository: market_data_poc

Overview

This Proof of Concept (PoC) demonstrates a simple pipeline for fetching stock market data from an API, transforming the data, and saving it in both CSV and Excel formats. The project is designed to be lightweight and efficient.

Folder Structure

  • config/: Reserved for configuration files (e.g., API keys).
  • data/: Contains output CSV and Excel files.
  • logs/: Tracks logs for each pipeline run.
  • notebooks/: For experimentation and exploratory data analysis.
  • scripts/: Core pipeline logic for fetching and transforming data.
  • tests/: Unit tests for the project.

Setup Instructions

  1. Clone the repository using git clone.
  2. Create a virtual environment with python3 -m venv venv.
  3. Activate the environment and install dependencies with pip install -r requirements.txt.
  4. Run the pipeline using python main.py.

API Configuration

The project uses the Marketstack API to fetch stock market data. Ensure you have an API key configured in config/config.py.

Technologies Used

  • Python (pandas, requests, logging)
  • Data storage formats: CSV, Excel
